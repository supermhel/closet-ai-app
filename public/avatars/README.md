# Avatar Images

This directory should contain avatar images for users to choose from:

Required avatar image files:
- avatar1.png
- avatar2.png
- avatar3.png
- avatar4.png
- avatar5.png
- avatar6.png

Example avatar images should be:
- 200x200 pixels or larger (square)
- With different styles/characters/themes
- Preferably with transparent backgrounds for a clean look

These images will be displayed in the AvatarSelector component for users to choose their profile picture.
